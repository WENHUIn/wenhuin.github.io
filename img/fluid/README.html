快速开始

1. 搭建 Hexo 博客

如果你还没有 Hexo 博客，请按照 Hexo 官方文档 进行安装、建站。

2. 获取主题最新版本

请优先下载 最新 release 版本，master 分支无法保证稳定。

下载后解压到 themes 目录下并重命名为 fluid。

3. 指定主题

如下修改 Hexo 博客目录中的 _config.yml：

theme: fluid  # 指定主题

language: zh-CN  # 指定语言，可不改

4. 创建「关于页」

自 v1.7.0 开始，「关于页」需要手动创建：

$ hexo new page about

创建成功后，编辑博客目录下 /source/about/index.md，添加 layout 属性。

修改后的文件示例如下：

title: about
date: 2020-02-23 19:20:33
layout: about
---

这里写正文，支持 Markdown, HTML

功能特性

  • 图片懒加载
  • 自定义代码高亮方案
  • 内置多语言
  • 支持多款评论插件
  • 支持使用数据文件存放配置
  • 自定义静态资源 CDN
  • 内置文章搜索
  • 页脚备案信息
  • 网页访问统计
  • 支持 LaTeX 数学公式
  • 音乐播放器
